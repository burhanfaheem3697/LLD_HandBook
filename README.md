# Software Design & Low-Level Design Mastery Handbook

This folder contains a professional Markdown curriculum for Java-centered software design and low-level design mastery.

Start here:

1. `00_BOOK_SPEC.md`
2. `01_CURRICULUM_OVERVIEW.md`
3. Phase files `02_` through `19_`
4. `20_PROJECT_PORTFOLIO.md`
5. `21_ASSESSMENT_SYSTEM.md`
6. `22_RESOURCE_CATALOG.md`

The curriculum uses a spiral model: every major idea appears first as a programming technique, then as an object model, then as a design principle, then as a framework mechanism, and finally as an architectural trade-off in production systems.

Each phase file now includes a `Guided Self-Study Program` section with lesson-by-lesson schedules, labs, machine-coding exercises, diagrams, revision plans, common mistakes, design-thinking prompts, open-source connections, and checklists.
